Before enabling auto-updates

  • OS updates use apt upgrade — safe mode, never removes packages.
  • Auto-reboot will reboot the host immediately after the update if a reboot is required. Enable only on hosts that can tolerate unplanned restarts.
  • Docker stack redeploys pull the latest image tag — pin your image versions if a breaking change would be disruptive.
  • Sudo password must be stored in Admin → Hosts → Credentials for OS updates on non-root users.
  • All schedules are in UTC.
