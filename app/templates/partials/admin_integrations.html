OPNsense Optional
{% if integ.opnsense_configured %} Configured {% else %} Not set up {% endif %}

Monitor firewall and plugin updates on OPNsense via its API.

How to create an OPNsense API key
  1. Log in to OPNsense → System → Access → Users
  2. Edit your admin user → scroll to API keys → click +
  3. Download the generated key file — it contains both Key and Secret
  4. Grant the user the Firmware privilege under System → Access → Users → Edit → Effective Privileges

Home Assistant Optional
{% if integ.ha_configured %} Configured {% else %} Not set up {% endif %}

Monitor Home Assistant core and add-on updates. Keepup can notify you when updates are available but cannot install them automatically.

How to create a Long-Lived Access Token
  1. Open Home Assistant in your browser
  2. Click your username (bottom-left) → Security tab
  3. Scroll to Long-Lived Access Tokens → click Create Token
  4. Give it a name like keepup and copy the token

Portainer Docker management API
{% if integ.portainer_configured %} Configured {% else %} Not set up {% endif %}

Portainer is a web UI for managing Docker on your home servers. Keepup uses its API to detect when your container images have updates available. Only needed if you run Docker containers.

How to find these values in Portainer
  1. Open Portainer (usually https://your-server-ip:9443)
  2. Click your username (top-right) → Account Settings
  3. Scroll to Access Tokens → Add access token
  4. Name it keepup — copy the token before closing

Include https:// and the port (usually 9443)

Docker Hub Optional
{% if integ.dockerhub_configured %} Configured {% else %} Not set up {% endif %}

Without an account, Docker Hub limits container image checks to 100 per hour shared across your IP. With a free account you get 200 per hour just for you.

How to create a Docker Hub access token
  1. Sign in at hub.docker.com (free account is enough)
  2. Click your username → Account Settings → Personal access tokens
  3. Click Generate new token, name it keepup, set access to Public Repo Read-only
  4. Copy the token — you won't see it again

Use an access token, not your password. Tokens can be revoked individually.
